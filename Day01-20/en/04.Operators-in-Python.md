## Day 4 - Operators in Python

### Arithmetic operators
```python
a, b = 10, 3
print(a + b)  # 13
print(a - b)  # 7
print(a * b)  # 30
print(a / b)  # 3.333...
print(a // b) # 3
print(a % b)  # 1
print(a ** b) # 1000
```

### Assignment operators
```python
x = 5
x += 2  # 7
x *= 3  # 21
```

### Comparison operators
`==`, `!=`, `>`, `<`, `>=`, `<=`

### Logical operators
`and`, `or`, `not`

### Bitwise operators (advanced)
`&`, `|`, `^`, `~`, `<<`, `>>`

Use parentheses to improve readability when expressions are long.
