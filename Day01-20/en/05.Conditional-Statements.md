## Day 5 - Conditional Statements

### Basic `if`
```python
score = 85
if score >= 60:
    print('Pass')
```

### `if-else`
```python
age = 17
if age >= 18:
    print('Adult')
else:
    print('Minor')
```

### `if-elif-else`
```python
score = 92
if score >= 90:
    print('A')
elif score >= 80:
    print('B')
elif score >= 70:
    print('C')
else:
    print('D')
```

### Good practices
- Keep conditions simple.
- Avoid deep nesting when possible.
- Write clear boundary conditions (`>=`, `>`, etc.).
