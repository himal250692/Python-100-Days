## Day 9 - Common Data Structures: List (Part 2)

### Add elements
```python
langs = ['Python', 'Java']
langs.append('C++')
langs.insert(1, 'SQL')
```

### Remove elements
```python
langs.remove('Java')
last = langs.pop()      # remove and return last element
langs.clear()           # remove all elements
```

### Search and count
```python
items = ['a', 'b', 'a']
print(items.index('b'))
print(items.count('a'))
```

### Sort and reverse
```python
nums = [5, 2, 9, 1]
nums.sort()                 # ascending
nums.sort(reverse=True)     # descending
nums.reverse()              # reverse current order
```

### List slicing
```python
arr = [0, 1, 2, 3, 4, 5]
print(arr[1:4])   # [1, 2, 3]
print(arr[::2])   # [0, 2, 4]
```
