## Day 3 - Variables in Python

### Variables
A variable stores a value and gives it a name.

```python
name = 'Alice'
age = 20
height = 1.68
is_student = True
```

### Naming rules
- Use letters, numbers, and underscore.
- Cannot start with a number.
- Case-sensitive (`age` and `Age` are different).
- Do not use Python keywords (like `if`, `for`, `class`).

### Basic data types
- `int`: integers
- `float`: decimal numbers
- `str`: text
- `bool`: `True` / `False`

### Input and output
```python
name = input('Enter your name: ')
print('Hello,', name)
```

### Type conversion
```python
age = int(input('Enter age: '))
price = float('19.99')
```
